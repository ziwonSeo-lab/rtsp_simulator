# RTSP Client Module Documentation

이 디렉토리는 RTSP Client Module의 문서와 다이어그램을 포함합니다.

## 📁 문서 구조

- `architecture.md` - 시스템 아키텍처 및 프로세스 흐름도
- `README.md` - 이 문서

## 🎯 주요 다이어그램

### 전체 시스템 흐름

**RTSP 스트림들** → **캡처** → **블러링** → **저장** → **파일 이동** → **완료**

각 단계마다 **멀티프로세스**로 병렬 처리하며, **실시간 통계**를 지속 수집합니다.

## 📋 핵심 특징

- 🔄 **공유 워커 아키텍처**: 각 워커가 모든 스트림을 순환 처리
- 🚀 **멀티프로세싱**: CPU 코어 활용 극대화
- 💾 **2단계 저장**: SSD→HDD 최적화된 저장 시스템
- 📈 **실시간 모니터링**: 리소스 및 성능 지속 추적
- 🔧 **모듈화 설계**: 독립적인 컴포넌트들로 구성

자세한 내용은 `architecture.md`를 참조하세요.
